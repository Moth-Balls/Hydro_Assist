# Hydro Assist

This a framework for an autonomous hydroponic management system.

## Features

- pH, EC, and temperature monitoring of system's resevoir
- Automatic dosing of [General Hydroponics FloraSeries](https://generalhydroponics.com/products/floraseries/) nutrients
- Automatic dosing of pH up/down solutions
- Model Predictive Control (MPC) for calculation dosing based on various plant species
- Front-end web interface for live monitoring and control as well as choice of plant profiles
